Technical Summary

For a detailed description of the dictyBase architecture, please see the detailed architecture page

dictyBase is run on an Ubuntu Linux 6.06 Server running Apache and the programming language is Perl.

The database software is Oracle 10g Enterprise Server.

The database schema is a modified version of the Chado Database Schema.

The Genome Browser is an implementation of GBrowse from the Generic Model Organism Database Construction Set (GMOD).

The DHTML dropdown menus seen on every page are licensed from Milonic.
