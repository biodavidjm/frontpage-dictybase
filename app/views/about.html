We dictyBase

Technical Summary

This beta version of dictyBase was built using AngularJS, with the lastest markup (HTML5) and style (CSS3) language versions.

Bootstrap is the framework used to develop the responsive features.

The architecture is hosted entirely on a cloud system. The applications are built and run on docker containers.
