Bioinformatics Resources

  • AmiGO: Web-based browser for The Gene Ontology
  • Apollo: A Java-based application for annotating genomic sequences
  • Bioinformatics.ca: Canadian Bioinformatics Resources
  • BRENDA: The Comprehensive Enzyme Information System
  • ExPASy: The Expert Protein Analysis System proteomics server of the Swiss Institute of Bioinformatics dedicated to the analysis of protein sequences and structures as well as 2-D PAGE
  • International Society for Biocuration (ISB): Information and meetings relating to curation of biological data
  • Chado: The GMOD database schema
  • Gene Ontology (GO) Consortium: The GO project provides a controlled vocabulary to describe gene and gene product attributes in any organism
  • GBrowse: The Generic Genome Browser: A combination of database and interactive web page for manipulating and displaying annotations on genomes
  • GMOD: Generic software components for model organism databases
  • KEGG: Links to Pathway and Other Databases
  • InParanoid: Database of eukaryotic ortholog groups
  • Open Biomedical Ontologies: Umbrella web address for well-structured controlled vocabularies for shared use across different biological and medical domains, includes the Dictyostelium anatomy ontology
  • National Center for Biomedical Ontology: Advancing biomedicine with tools and methodologies for the structured organization of knowledge
  • UniProt: Universal Protein Resource
